# portfolio

Terminal‑inspired portfolio built with React, Three.js, and SCSS.

## dev

```bash
npm run dev       # → localhost:3000
npm run build     # → dist/
npm run preview   # preview the build
```

## stack

- **React 18** + **Vite**
- **Three.js / R3F** — 3D globe, scene transitions
- **SCSS** — global styles, no CSS modules (except navbar)
- **react-icons** — Feather Icons + dev icons
